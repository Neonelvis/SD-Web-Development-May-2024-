Text Formatting

You can make text bold using the <b> tag or strong using the <strong> tag

You can italicize text using the <i> tag or emphasize it using the <em> tag, like this.

You can also highlight text using the <mark> tag

You can underline text using the <ins> tag

You can use <sub> tag for subscript, like H20, and <sup> tag for superscript, like x2.

Character Entities

You can use the <code> tag to display code snippets inline, like console.log('Hello, Everyone!');

For longer code blocks, you can use <pre> tag, which preserves white spaces formating
